Introduction

House Automation System (HAS) is an application designed for users who would like to control some features of their homes, such as temperature, light, and more. This application is being developed by a group of computer scientists from the University of Aveiro in a project of the chair IES

Description

House Automation System (HAS) is about a application to control your house. This means thats you have control of some features as luminosity, movement, temperature, etc.. In this product we are using as Raspberry Pi for control of the sensors and comunications with your compouter or mobile phone, luminosity, movement, temperature sensors, and a IKEA tradfri
